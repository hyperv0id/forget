# 【杀戮尖塔】角色模组设计白皮书：The Forget (v0.04)

**英文名：** The Forget
**Slogan：** Forget for Get (遗忘是为了获得 / 得失之间)
**版本：** v0.04（Pre-Dev Final）
**原作背景：** 《明日方舟》岁系列角色 - 望（Wang）
**核心身份：** 第181枚神识碎片（黑子）

---

## 一、核心设计愿景 (Core Vision)

### 1.1 高概念
玩家扮演的不是“望”本人，而是为了屠神而切分出的**神识碎片**。这是一场在棋盘上孤独行进的旅程：
*   **卡牌是记忆**：有些记忆用来回味（修复），有些记忆用来透支（模仿）。
*   **战斗是落子**：在“体面”与“无耻”之间寻找胜机。

### 1.2 核心冲突
*   **资源冲突：** 你需要【名望】来维持体面（清理污染），但往往需要透支【名望】来换取爆发。
*   **关系冲突：** 你需要【模仿】兄弟姐妹的权能来获胜，但每一次模仿都是对他们“道”的磨损与曲解。
*   **自我冲突：** 你是想做一个完美的复制品，还是一个满身伤痕的弑神者？

## 二、基础资源系统 (Resource System)

### 2.1 【自我】 (Self / HP)
*   **定义：** 传统的生命值。
*   **特殊性：** 是所有透支行为的最终兜底支付手段。当名望和好感度无法支付代价时，往往由 HP 偿还。

### 2.2 【名望】 (Reputation) - “灯火机制”
名望是一条**跨战斗的全局环境数值**（类似《暗黑地牢》的光照度），有正负之分，决定了玩家的**生存模式**。

*   **名望 > 0（体面态 / High Light）：**
    *   **环境特性：** 可以消耗名望，清理手牌中的【污染】（诅咒/状态）。
    *   **清理动作：** 玩家可主动消耗 `N` 点名望，将手牌中的【污染】（诅咒/状态）移除本场战斗。这是清理卡组的主要手段。
    *   **代价：** 战斗节奏偏慢，侧重运营与控制。
  
*   **名望 < 0（无耻态 / Low Light）：**
    *   **环境特性：** 无法用名望清理【污染】，【污染】牌将滞留在卡组中。
    *   **数值修正：** 获得**全局伤害加成**（越堕落越强）。
    *   **生存逻辑：** **DPS Check（伤害检测）**。玩家必须在卡组被【污染】彻底淹没窒息之前，利用高额伤害爆发解决敌人。
    *   **舆论隐喻：** “在被骂死之前，先打赢你。”

### 2.3 【好感度】 (Affinity) - “不可见的弹簧”
*   **定义：** 按对象（如：令、年、夕等）分别结算的隐藏数值。
*   **可见性：** **UI 不显示具体数值**。玩家只能通过卡面特效（表情/裂纹）和动态关键词来判断当前关系状态。
*   **经济循环：**
    *   **消耗端：** 使用**【模仿】**类卡牌（借用权能）会降低对应对象的好感度。
    *   **回复端：** 使用**【眷恋】**类卡牌（回忆过往）可提升好感度。
*   **防刷机制（Anti-Stalling）：** 所有能提升好感度的【眷恋】牌，均带有 **`Exhaust`（消耗）** 属性。记忆是有限的，不可在单场战斗中无限重温。

### 2.4 累计反噬次数 - “过程罪孽”
*   **定义：** Act 4 结局判定的核心参数。
*   **计算公式：** **过程累加制**。
    *   `累计反噬次数` = 本局游戏中，因好感度过低而触发“模仿不稳定/反噬”效果的**累计总次数**。
*   **设计意图：** 即使你在 Boss 战前突击修复了关系（把好感刷满），你曾经对他人的“过度利用”和“曲解”依然被系统记录在案。你无法洗白你的罪孽。

## 三、卡牌系统：记忆的具象

### 3.1 卡牌分类与功能

| 类型 | 颜色/视觉 | 核心功能 | 对应资源逻辑 |
|:---|:---|:---|:---|
| **模仿 (Imitate)** | 渐变/多态 | 借用兄弟姐妹权能，强力效果 | **消耗好感度**。好感越低，副作用越强（卡面变暗/碎裂）。 |
| **眷恋 (Cherish)** | 暖色调 | 修复关系、过牌、防御 | **回复好感度**。**必带消耗属性**（不可循环利用）。 |
| **权谋 (Scheme)** | 暗金/灰色 | 操控名望、利用污染 | **消耗/产出名望**。连接“体面”与“无耻”的桥梁。 |
| **落子 (Move)** | 纯黑 | 终结技、禁忌爆发 | 造成巨额伤害或特殊胜利，但留下长期高额代价（如扣除名望上限）。 |
| **污染 (Pollution)** | 墨黑/噪点 | 干扰手牌 | **被操作对象**。体面时可用名望清理，无耻时作为死线倒计时。 |

### 3.2 动态卡面与交互 (UI/UX)
为了解决隐藏数值的不可读问题，采用以下标准：

1.  **动态卡面：** 【模仿】牌的立绘根据好感度分级变化（微笑 -> 沉默 -> 阴影 -> 破碎）。
2.  **动态文本（Hover Tip）：**
    *   当玩家将鼠标悬停在【模仿】牌上时，提示栏（Tooltip）会根据当前隐藏好感度，**实时显示**附加的红字负面效果。
    *   *例：当前对“令”好感极低，悬停【模仿：令】显示：“<不稳定>：打出后，将在弃牌堆生成2张晕眩。”*

## 四、玩法阶段与进程 (Progression)

### 4.1 前期 (Act 1-2)：建立账本
*   玩家通过【模仿】牌获得超模战力，轻松过关。
*   【眷恋】牌充裕，好感度容易维持平衡。
*   玩家开始学习用【名望】清理【污染】，维持卡组纯净。

### 4.2 中期 (Act 3)：资源崩坏
*   【眷恋】牌（消耗品）逐渐耗尽，或跟不上高频战斗的消耗。
*   玩家被迫面临抉择：
    *   **A. 节制路线：** 减少模仿，依靠自身能力刮痧，尽量减少反噬触发次数。
    *   **B. 堕落路线：** 不顾好感破裂，强行使用模仿（触发反噬），甚至主动进入“无耻态”利用高伤速杀。
*   **污染危机：** 随着名望波动，卡组可能会一度被污染填满，考验玩家的生存极限。

### 4.3 后期 (Act 4)：镜像审判
进入【天元空间】，面对特殊机制 Boss。

*   **Act 4 精英：【望 - 空壳】**
    *   **机制：** 镜像战。
    *   **强度挂钩：** Boss 的数值强度直接乘以你的 **`累计反噬次数`**。你一路走来触发的反噬越多，镜像就越强壮、越狰狞。
  
*   **最终 Boss：【岁 - 诸相非相】**
    *   **神技系统：** 针对你的资源短板进行打击（压HP、压名望、压牌库）。
    *   **终局博弈：** 此时通常已无任何【眷恋】牌可用，玩家必须用尽最后一点名望和HP，在自我毁灭前完成屠神。

## 五、结局分支 (Endings)

结局由 **【是否击败岁】** + **【最终名望状态】** + **【累计反噬次数】** 共同决定。

1.  **【棋碎人亡】**：战斗失败。
2.  **【胜天半子】（无耻/反噬次数极高）：** 击败岁，但名望为负，反噬次数极高。
    *   *叙事：* 你赢了，但你成为了比神更可怕的怪物。兄弟姐妹视你为仇寇。
3.  **【孤宿无两】（体面/反噬次数低）：** 击败岁，名望为正，反噬次数低。
    *   *叙事：* 完美的胜利，但为了保持这份完美，你耗尽了所有。你选择自我放逐。
4.  **【众生弈】（真结局 - 动态平衡）：** 击败岁，名望体面，反噬次数中等（有过借力但未失控）。
    *   *叙事：* 承认了自己的平庸与他人的重要。得失之间，方为众生。

## 六、技术实现关键点 (Tech Checklist)

*   [ ] **ResourceManager：** 实现双向名望轴，以及名望 < 0 时的全局伤害修正（Damage Multiplier）。
*   [ ] **ActionManager：** 实现UI上的“拖拽卡牌到名望槽”或点击按钮进行清理动作（消耗名望移除手牌中的污染/状态牌）的逻辑。
*   [ ] **AffinityManager：**
    *   实现 `Map<CharacterEnum, int>` 存储好感度。
    *   实现监听器：当【模仿】打出时扣除，当【眷恋】打出时增加。
    *   实现 Act 4 的“累计反噬次数”计数器（仅增不减）。
*   [ ] **CardBase：**
    *   重写 `applyPowers()` 和 `calculateCardDamage()`，使其能读取 AffinityManager 并动态修改卡牌描述（Description）。
    *   实现动态卡面（img path switch）。
