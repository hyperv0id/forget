# 【杀戮尖塔】角色模组设计白皮书：The Forget (v0.06)

> ⚠️ **更新（2026-01-13）**：本版本主要更新【名望】的分段区间口径，引入 `0 ~ -3` 的缓冲/透支区，并明确“极低名望”才会触发战斗副作用。

**英文名：** The Forget  
**Slogan：** Forget for Get (遗忘是为了获得 / 得失之间)  
**版本：** v0.06（Reputation Tier Revision）  
**设计日期：** 2026-01-13  

---

## 1. 改动目标（为什么要改）

旧口径里，“名望一旦为负就开始承担战斗副作用”，会导致两个问题：

1. **玩法节奏过硬：** 玩家一旦跌破 0 就被迫进入“拼命模式”，缺少缓冲窗口。
2. **理解成本过高：** “能不能清理污染”“好感度结算效率”“模仿是否吃血”等规则分散在多处，需要一个统一的直觉分段。

v0.06 的目标是：把名望做成**有清晰区间语义的环境变量**——玩家只需看当前区间，就能知道自己处在“运营 / 正常 / 缓冲 / 拼命”哪种状态。

---

## 2. 名望分段区间（当前口径）

> 说明：名望数值**不封顶**（不做硬限制），但玩法效果在这 4 档上呈现“台阶式”变化；超出 `+6` / `-6` 的数值属于“同档延伸”，不会额外开启新档位。  
> **默认平衡区间：** 以 `[-6, +6]` 作为“通常可达范围”的调参锚点（不是硬上限）。

| 区间 | 数值范围 | 清理污染 | 好感度获取 | 战斗特性 | 玩家直觉定位 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **高名望** | **+4 ~ +6（+4 起同档）** | **自动/免费** | **双倍效率** | 无 | **完美运营**：快速积累好感，卡组极度干净。 |
| **正名望** | **+1 ~ +3** | **可用** | **正常** | 无 | **正常状态**：可以花名望去洗牌，正常玩。 |
| **负名望** | **0 ~ -3** | **不可用** | **减半/变慢** | 无 | **缓冲/透支**：不能洗牌，但还没有副作用；给玩家一个**喘息区**。 |
| **极低** | **-4 ~ -6（≤ -4 起同档）** | **不可用** | **停止** | **模仿牌耗血 + 增伤** | **拼命/爆发**：放弃运营，拿血换伤害。 |

---

## 3. 规则落点（系统实现应该怎么接）

### 3.1 污染清理

- **正名望（+1~+3）：** 玩家可主动消耗名望进行清理（交互/UI 层面需要提示名望不足/不可用原因）。
- **高名望（+4 起）：** 清理变为自动/免费（具体交互形态可以是“被动净化”或“清理按钮不消耗名望”，实现时二选一，但 UI 必须直观）。
- **负名望/极低（≤0）：** 清理不可用，污染压力成为卡组/节奏的倒计时。

### 3.2 好感度获取

- 高名望：**2x**
- 正名望：**1x**
- 负名望：**0.5x**
- 极低：**0x（停止）**

> 注意：这里只约束“好感度上升/获取”的倍率；好感度的扣减（例如模仿的直接扣减）不在此倍率内。

### 3.3 极低名望的战斗副作用

- 仅在 **≤ -4** 触发：
  - 模仿牌额外耗血（Lose HP，不计护甲）
  - 模仿牌获得增伤（增伤数值为后续调参项，当前口径只要求有明确钩子）

---

## 4. UI 口径（玩家应当看到什么）

顶栏名望 tooltip 至少要包含：

1. 当前名望数值（正数建议显示 `+N`）
2. 当前区间名称（高名望/正名望/负名望/极低）
3. 三行摘要（清理污染/好感度获取/战斗特性）

---

## 5. 验收口径（实现完成的定义）

- 名望允许负值、可存档读档
- 分段规则在代码层有“统一入口”（供污染/好感/模仿系统复用）
- Roadmap 与 UI 文案不再出现旧口径（例如 “名望 < 0 立刻触发模仿扣血”）
